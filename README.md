# 🎨 UI来找茬 (UI 还原度走查工具)

> 一个专为前端开发和 UI 设计师打造的轻量级 macOS 工具。
> A lightweight, pixel-perfect review tool for frontend developers.


## ✨ 简介 (Introduction)

在开发过程中，你是否为了对比 UI 设计稿和实际页面的还原度，频繁切换窗口？
**UI来找茬** 帮你解决这个问题！它是一个**置顶悬浮窗**，你可以调节透明度，让设计稿直接“叠”在你的代码实现上，像素级校对，一眼看出差异。

## 🚀 核心功能 (Features)

- 🖼 **拖拽上传**：直接把设计稿图片拖入窗口即可显示。
- 💧 **透明度调节**：通过滑动条精准控制透明度，轻松对比底层页面。
- 📌 **永远置顶**：窗口始终悬浮在最上层，不会被浏览器遮挡。
- 🖱 **鼠标穿透**：开启“穿透模式”后，你可以透过设计稿直接操作底下的网页（点击、滚动），完全不影响开发。
- 🔒 **本地安全**：所有图片仅在本地内存处理，无需联网，绝对安全。

## 🛠 安装与使用 (Installation)

1. 在 [Releases](https://github.com/jjjjtian-coder/ui-diff-tool/releases) 页面下载最新的 `.dmg` 安装包。
2. 安装并打开应用。
3. 将你的 UI 设计稿拖入窗口。
4. 调整透明度，开始像“透写台”一样校对你的页面吧！

## 💻 技术栈 (Tech Stack)

- **Electron**
- **HTML5 / CSS3 / JavaScript**
- **Node.js**
